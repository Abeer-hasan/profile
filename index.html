  • Project 1: Aircraft Engine Remaining Useful Life Prediction

    The aim of this project was to use historical data to develop health assessment and prediction models that can reliably anticipate the number of flights remaining for the 100 aircraft engines that are currently being examined. A huge amount of historical data (data from 24 sensors) from over 260 engines was available, covering their entire life cycle. The sensor data was divided into six regimes based on the environment in which the aircraft engine was operating (as determined by altitude, Mach number, and throttle/power setting). To deal with the issue of regime separation, all of the data was initially split into 6 regimes based on the available data and normalized between 0 and 1. We then performed a feature selection using fisher criteria and used logistic regression to plot a degradation curve. We calculated Euclidian distance between one test data and 260 other training data and nine trajectories were selected based on the best matching curves. We take the median of the nine values as the last flight number. The difference between the last flight number and the current flight number is the Remaining Useful Life.
